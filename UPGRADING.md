# Upgrading Guide

This document highlights breaking changes between published versions of `@schie/epc`.

## Current status

There are no published breaking upgrades yet. For pre-1.0 releases, assume APIs can change and review release notes before upgrading.

## When this will be updated

Once a breaking change lands, this document will include:

- version-to-version upgrade steps
- renamed or removed exports
- behavioral changes in EPC encoding/decoding rules
